# Java Final Project - UNO Game

Java (Spring Boot) と React (TypeScript) を使用して構築された、リッチな演出と独自のルールを備えた本格的な UNO ゲームアプリケーションです。

## 🎮 ゲームの特徴

*   **リッチな演出**:
    *   **UNO宣言演出**: 残り1枚になった瞬間、ド派手な「UNO!」アニメーションが画面中央に出現。
    *   **手札交換 (SWAP)**: SWAPカード使用時、画面全体を使った「HANDS SWAPPING!」演出が発生。
    *   **勝利画面**: 勝者が決まると中央に勝利ウィンドウが表示されます。
*   **カスタムルール**:
    *   **初期手札**: スピーディーな展開のため、**4枚** スタート。
    *   **UNOペナルティ**: 残り1枚になってから **1秒以内** に「UNOボタン」を押さないと、ペナルティ（+1枚）が発生するハードモード仕様。
*   **CPU対戦**:
    *   3人のCPU対戦相手（弱体化調整済み：ランダム思考で初心者にも優しい設計）。
    *   4人プレイ（プレイヤー1人 vs CPU 3人）。
*   **デザイン**:
    *   POPで視認性の高いカードデザイン（裏面はシンプルな白ロゴ）。
    *   レスポンシブで画面端に寄り過ぎない見やすいレイアウト。

## 🛠 技術スタック

### Backend (サーバーサイド)
*   **Java 17+**
*   **Spring Boot 3.x**: Web APIサーバーとして機能。
*   **Clean Architecture**: ドメインロジック（ルール）とインフラストラクチャ（API）を分離した設計。
*   **Maven**: ビルドツール。

### Frontend (クライアントサイド)
*   **Rules**:
    *   React
    *   TypeScript
    *   Vite
*   **Styling**:
    *   Pure CSS / CSS Modules (リッチなアニメーション @keyframes を多用)

## 🚀 実行方法

### 1. バックエンドの起動
`backend` ディレクトリで以下のコマンドを実行します。
```bash
cd backend
./mvnw spring-boot:run
```
サーバーが `http://localhost:8080` で起動します。

### 2. フロントエンドの起動
`frontend` ディレクトリで以下のコマンドを実行します。
```bash
cd frontend
npm run dev
```
ブラウザで表示されたURL（通常 `http://localhost:5173`）にアクセスしてゲーム開始！

## 📁 ディレクトリ構成

*   `backend/`: Spring Boot アプリケーション
*   `frontend/`: React アプリケーション

## ⚠️ ルール詳細ノート
*   **UNOボタン**: 自分の手札が残り1枚になった瞬間、画面中央にボタンが出現します。1秒以内に押してください。
*   **SWAPカード**: このカードを出すと、次のプレイヤーと手札を強制交換します（CPUはランダムに色を指定）。
